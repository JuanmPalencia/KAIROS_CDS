# KAIROS CDS — Emergency Fleet Management Digital Twin

**v1.0.0** | Enterprise-grade fleet management with AI, blockchain auditing, and advanced security.

---

## 🚀 Quick Start

```bash
python3 run_all.py              # Start (keep data)
python3 run_all.py --reset      # Full reset (delete all data)
python3 run_all.py --security   # Run cybersecurity scan
python3 run_all.py --logs       # View live logs
python3 run_all.py --stop       # Stop services
```

**That's it!** Everything is automated in one script.
